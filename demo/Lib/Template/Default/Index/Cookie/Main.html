Cookie服务

{include DemoBtn}
在TangFramework中，使用Cookie非常简单！
1、在Web控制器中调用$this->request->cookie()
2、通过Cookie的Service获取Cookie对象。使用\Tang\Web\Cookie::getService()
通过上述两种方法可获得\Tang\Web\Cookie\ICookie接口对象。然后使用该对象进行cookie的读写删除操作

Cookie的配置

Cookie的配置文件是Cookie.conf文件，下面是默认配置。
当使用set($name,$value,$expire=-1,$path='',$domain='');expire path domain都为默认值的时候，将使用配置文件中的值进行设置。 
							{
		"expire" : "86400", //Cookie的时效
		"path" : "",//Cookie生效路径
		"domain" :"",//域
		"prefix" : "tang-"//Cookie前缀
}

基本实例

							$cookie = \Tang\Web\Cookie::getService();
$cookie->set('name','value');
echo $cookie->get('name');
$cookie->delete('name');
$cookie->clean();
