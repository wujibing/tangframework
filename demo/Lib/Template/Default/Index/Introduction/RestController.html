RestController

RestController提供Rest风格的控制器，RestController继承于WebController,于WebController的Ajax不同，RestController默认支持Ajax,并且是根据后缀名来进行返回。 例如： /index/index/main.json[返json] /index/index/main.xml[返xml] /index/index/main.yaml[返yaml] 如果没有后缀名或者后缀名不同于json xml yaml则返回html
