日志服务

日志服务暂时提供文件,数据库驱动，后面会提供更多的日志存储驱动。

基本实例

\Tang\Log\LogService::getService()->write('我是日志消息','ERR');
\Tang\Log\LogService::getService()->write('我是日志消息',\Tang\Log\LogService::CRIT);
...其余操作查看接口

配置 Log.conf

{
    "record":true,//默认开启 填写flase的话则不记录
    "recordLevel":
    [
        "MERG","ALERT","CRIT"//只记录"MERG","ALERT","CRIT"类型日志
    ],//允许记录的日志级别 可用值MERG ALERT CRIT ERR WARNING NOTIC INFO DEBUG SQL   为空的时候记录所有的级别 E
    "defaultDriver":"file",//默认的驱动 支持file database
    "file":
    {
        "directory":"Log",//日志的目录，位于Application配置下dataDirctory设定下。
        "filePath":"%Y-%m/TangLog-%y-%m-%d.log"//日志文件名格式  %后面加上date的日期格式就好了
    },
    "database":
    {
        "tableName":"log",//表名
        "source":"" //使用的数据源
    }
}
