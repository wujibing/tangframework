CliController

CliController是Cli模式下的控制器。CliController没有视图。CliController通过CliRequest和CliResponse进行输入输出。 例如： 
    public function main()
    {
        $response = $this->request->getResponse();
        $response->writeLine('Hi.请问您叫什么名字？');
        $name = $this->request->readLine();
        $response->writeLine(sprintf('哦。原来您叫:%s;请问您的性别是(1为男 2为女)：',$name));
        $sex = $this->request->readAChar() == 1 ? '先生':'女士';
        $response->writeLine(sprintf('欢迎您:%s%s',$name,$sex));
    }
