文件服务

文件服务提供了一系列的读写操作，详情操作参阅接口

基本实例

$file = \Tang\Services\FileService::getService();
$file->create('/data/www/test/a.txt');//创建个文件
$content = 'hello';
$file->append('/data/www/test/b.txt',$content);//向b.txt追加$content内容
$file->write('/data/www/test/c.txt',$content);//向c.txt写入$content
$file->read('/data/www/test/d.txt',$content);//将d.txt内容读入$content
...其余操作查看接口
