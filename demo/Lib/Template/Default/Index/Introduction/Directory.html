目录服务

目录服务提供了一系列的创建/删除/移动等操作，详情操作参阅接口

基本实例

$directory = \Tang\Services\DirectoryService::getService();
$directory->copy('/data/www/a/','/data/www/b/');//将a复制到b
$directory->create('/data/www/c');//创建/data/www/c文件夹
$directory->delete('/data/www/c');//删除目录
