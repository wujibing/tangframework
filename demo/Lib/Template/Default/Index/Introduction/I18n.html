I18n服务

如何加入自己的语言包？

开发者可在网站目录Lib/I18n/语言/编码/下建立自己的语言包。例如建立User.php，里面的内容为返回一个数组。语言键对应值。例如'登录','hi!%s'=>'您好!%s')

基本实例

$I18n = \Tang\Services\I18nService::getService();
echo $I18n->get('welcome!%s',array('您的名字'));//输出welcome!您的名字
echo $I18n->get('user->login');//载入自定义的User语言包login项
echo $I18n->get('user->hi!%s',array('jack'));//输出您好!jack
...其余操作查看接口
