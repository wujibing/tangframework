Request服务

你造吗？

Tang\Services\RequestService::getService()在cli环境下获取的是CliRequester，web模式下获取的是WebRequester！框架里面的控制器都包含了Tang\Request\IRequest 接口对象。可通过控制器中request属性获取

Tang\Request\IRequest 接口源码

                    {$interfaceSourceCode}
Web模式下的Requester比Cli模式下多几个方法
getHttpMethod()获取客户端使用的 HTTP 数据传输方法（如 get、post 或 head）
clientIp() 获取客户端IP信息 返回\Tang\Web\Ip\IClientIp接口对象
cookie() 获取\Tang\Web\Cookie\ICookie接口对象
session() 获取\Tang\Web\Session\ISession接口对象
browser() 获取浏览器信息 返回\Tang\Web\Browser\IBrowser接口对象
getUrlReferrer() 获取HTTP_REFERER
