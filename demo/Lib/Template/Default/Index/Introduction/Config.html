Config服务

Config服务是一个特殊的服务！

他不继承ServiceProvider类和依赖配置文件！如果要使用自己的配置类时候，我们只要添加一个新类，实现Tang\Config\IConfig接口，在载入框架TangApplication.php后马上使用ConfigService::setConfigClass(配置类地址);即可

案例

$config = ConfigService::getService();
echo $config->get('charset','UTF-8');
echo $config->replaceGet('cache.*',array(......));
$config->set()
...其余操作查看接口
