Controller

Tang\Web\Controllers\Controller是所有控制器的基类！

Controller用于管理、创建控制器对象，并执行动作。并包含了$request、$parameters、$config、$i18n等公有属性！

如何建立一个控制器

控制器编写很简单，在网站目录Lib\Applications\模块名称\Controllers\模式[Cli|Web]\控制器名.php
在Cli模式下调用的控制器都位于Cli文件夹。Web模式都调用Web文件夹！

为了使得框架更简单，框架使用了包。所以我们在编写控制器的时候，也需要加入包名。
namespace Lib\Applications\模块名称\Controllers\模式[Cli|Web]; 也就是Lib的文件路径
use Tang\Web\Controllers\WebController;

class Cookie extends WebController{
public function sayHi()
{
echo 'Hi!';
}
}
sayHi就是动作名称，能被我们直接调用
