你造吗？

Token服务的结果取决于Crypt服务默认驱动的加密结果！

基本实例

$token = \Tang\Token\TokenService::getService();
$token->getInput('token',1000);//生产input控件的验证，并且只持续1000秒 。
$token->validate('token');//验证token令牌是否有效
$token->make('token',1000);生产token令牌值，有效期为1000秒
