数据库配置

说明

Tang的数据库很大部分借鉴于laravel的DB部分。非常感谢laravel！

数据配置位于Database.conf文件。 例如： 
{
            //这是一个数据源的对象，default为数据源名称 开发者可定义N个数据源。
            "default":
            {
                "charset":"utf8",//编码
                "collate":"",//排序规则
                "dbName":"test",//数据库名
                "tablePrefix":"cms_",//表前缀
                "driver":"mysql",//驱动 mysql pgsql 暂时支持这两种
                "persistent":1,//是否持久链接
                "readWriteSeparate"://是否读写分离
                {
                    "support":0, //1为支持
                    "masterNumber":2//写入服务器数量
                },
                "info"://这下面是服务器信息数组支持读写分离的话要填写多个服务器
                [
                    {
                        "host":"127.0.0.1",
                        "username":"root",
                        "password":"123",
                        "port":"3306",
                        "unixSocket":""//使用UNIXSOCKET连接
                    }
                ]
            }
    }
