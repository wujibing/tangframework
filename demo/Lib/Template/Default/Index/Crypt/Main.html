加密服务

Crypt管理器！

提供了aes、blowFish、loki97、thirdDes(3des)、twoFish、xTea、gost加密驱动。默认的驱动为aes。开发者可在配置文件配置加密默认驱动，方法和Cache配置一样

{include DemoBtn}

通过Tang\Crypt\CryptService服务可获取CryptManager,然后使用该对象进行加密解密操作

Crypt的配置

Crypt的配置文件是Crypt.conf文件，下面是默认配置。 
							{
    "defaultDriver":"aes", //aes为默认的驱动  支持aes blowFish loki97 thirdDes(3des) twoFish xTea gost加密方式
    "key":"123wcef", //密钥 当密钥和iv位数超过或不足的时候，框架会进行补位，具体查阅框架源码。
    "iv":"dsdss" //向量
}

基本实例

							$driver = CryptService::getService();
$driver->encode('xx')
$driver->decode('xx加密结果');
